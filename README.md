this is my e project which met all these requirements Problem Statement

Red Cards Limited is into selling cards for various occasions. They have a huge
collection of cards. They want to get on to internet with their collection, so that they
can reach various organizations and individuals. They want to host a website with all
the information about their organization and their collection of cards with the
snippets.
Requirement Specification:
The Web site is to be created based on the following requirements.
1) Home Page should contain logo of the site with proper sections and details of
cards available.
2) The site should display a menu which will contain the options for brief
introduction about the various categories and cards available, location of the
shop and any other information if required.
3) The information should be categorized according to the brand names of the
cards like if a User wants to see only “ARCHIES” cards or any other
company’s cards then he/she can click on a Link/button/menu etc and can
see only that Brand cards.
4) Another category option for cards of different occasions, like ‘Birthday’,
‘Anniversary’ etc. should be created and accordingly the cards should be
listed.
5) When a user selects any particular brand, a list of cards for that brand will be
displayed
6) A brief summary about individual cards should be displayed on the Web Page
along with the price.
7) There should be a “Contact Us” page which will have the Address of the Store
which is as follows and the mail address which when clicked will invoke the
local mail client from where they can send an email.
